🌾 Farm2Trade

Premium Agri-Logistics & Supply Chain Intelligence

📌 Overview

Farm2Trade is an enterprise-grade full-stack platform designed to bridge the gap between farmers, retailers, and transporters. It features AI-Driven Continuous Authentication and a Zero-Trust Security Infrastructure to ensure secure and transparent transactions in the agricultural ecosystem.

🚀 Key Capabilities

  • Continuous Behavior Analytics: Real-time risk assessment based on user interaction patterns.
  • Smart Marketplace: Dynamic bidding and secure crop auctioning for farmers and retailers.
  • Precision Logistics: Real-time GPS tracking and fleet management for transporters.
  • Weather Intelligence: AI-powered localized climate advice for optimal harvest planning.

🛠️ Tech Stack

Frontend React 19, Vite, Lucide, Recharts
Main Backend Spring Boot 3.x, Java 17, JPA
Security/ML Flask, Python, Scikit-Learn
Database MySQL, Hibernate
